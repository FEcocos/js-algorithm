# 효율성 (Efficiency)

효율적인 알고리즘 설계와 최적화 문제들을 모아놓은 폴더입니다.

## 포함되는 문제 유형
- 투 포인터 (Two Pointers)
- 슬라이딩 윈도우 (Sliding Window)
- 해시맵 활용
- 시간/공간 복잡도 최적화
